# Withreate-demo
Withreateのプロトタイプです。
 
# Requirement 
- @vitejs/plugin-vue@2.2.4
- autoprefixer@10.4.4
- firebase@9.6.9
- postcss@8.4.12
- tailwindcss@3.0.23
- vite@2.8.6
- vue-router@4.0.13
- vue@3.2.31

# Usage 
```bash
git clone https://github.com/shibata-kuronosuke/Withreate-demo.git
cd Withreate-demo
npm install
npm run dev
```
